데이터 중심, 사용자 중심의 새로운 보안

이제 보안도 SaaS로쉽고 간편하게

이제 보안이 달라지고 있습니다

점점 다양해지는 근무 환경에 맞춰 기업의 중요한 데이터 자산은클라우드에 저장되고 또 업무 편의성을 위해 다양한 SaaS로 손쉽게 복사되고 흩어집니다.



하지만 클라우드마다 다른 보안 환경으로 인해 표준화가 불가능하며 데이터 유출 사고의 경우 SaaS 공급자가 책임지지 않아 보안사고가 일어나면 기업이 중대한 피해를 입을 수 밖에 없습니다.



이제는 변화하는 업무 환경에 맞춰 기존의 네트워크 방식이 아닌, 사용자와 데이터 중심의 제로 트러스트 보안으로 정책이 변해야 합니다.

제로 트러스트 시대,스파이스웨어와 함께 신뢰를 만드세요

Spiceware One ZTS제로 트러스트 보안

중요한 데이터 자산이 어디에 얼마나 흩어져 있는지 한눈에 알 수 있도록 클라우드 스토리지와 SaaS로 이동하는 데이터를 분석하고 모니터링합니다. 또한 다양한 사용자 환경에서 안전하게 사내 시스템과 인터넷을 이용할 수 있도록 다중 인증(MFA) 및 싱글 사인 온(SSO) 등 기술을 지원합니다.

  • 데이터 모니터링
  • 데이터 암호화
  • 데이터 유출손실방지
  • 인증 강화
  • 사용자 정보 연동

Spiceware One DPS데이터 보호 서비스

데이터 내 개인·기밀 정보를 인식하고 분류하여 암호화하고 누가 어떤 정보에 접속했는지 실시간으로 모니터링하고 관리합니다. 또한 ISMS, CSAP 인증 기준 준수하여 상시 평가 대비 보고서 출력을 지원합니다.

  • 개인정보 암호화
  • 개인정보 접속 기록
  • 컴플라이언스 준수
  • 웹 민감정보 탐지 차단
  • 클라우드 서버 민감정보 관리
